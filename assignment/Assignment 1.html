5) What is the purpose of the "alt" attribute in the HTML "img" tag?

  • Provides a textual description of the image for users who cannot see it. This is crucial for people with visual impairments who use screen readers to browse the web.
  • Search engines use the alt attribute to understand the content of images. Descriptive alt text can help improve a page’s search engine ranking by making the content of the image searchable and relevant to the page’s context.
  • When an image fails to load (due to broken links, slow connections, etc.), the alt text is displayed in place of the image, providing context to the user.

8) What is the difference between HTML elements and HTML tags?

HTML tags are the fundamental building blocks of HTML that are used to create and define elements within a web page. They are the markup used to instruct the browser how to display the content. HTML elements are the complete set of tags and the content they enclose. An element includes both the start tag, the content, and the end tag.
